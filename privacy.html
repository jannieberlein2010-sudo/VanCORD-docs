VanCORD Privacy Policy

Last Updated: July 2025

VanCORD may store Discord user IDs, server IDs, channel IDs, role IDs, moderation records, ticket data, and server configuration settings required for bot functionality.

Collected data is used solely to provide moderation, logging, ticket management, verification, leveling, and server management features.

VanCORD does not sell, share, or distribute user data to third parties.

Server administrators may remove stored server data by removing the bot from their server.

By using VanCORD, you agree to this Privacy Policy.

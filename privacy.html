VanCORD may store Discord User IDs, Guild IDs, Channel IDs, Role IDs, moderation records, ticket data, verification data, leveling data, and server configuration settings required for the proper operation of the bot.

Stored information is used exclusively to provide moderation, logging, ticket management, verification, leveling, AutoMod, security, and server management features.

Data Sharing

VanCORD does not sell, rent, or share user data with third parties.

Server administrators may remove stored server data by removing VanCORD from their server or contacting support.

Reasonable measures are taken to protect stored data and prevent unauthorized access.
